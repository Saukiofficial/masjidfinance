# AI Agent System Prompt

Gunakan prompt master yang menginstruksikan AI membaca seluruh folder docs terlebih dahulu, mengikuti PRD, BUSINESS_RULES, DATABASE, PROJECT_STRUCTURE, CODING_STANDARD, UI_ADMIN, UI_LANDING_PAGE, dan ROADMAP. Gunakan Laravel 13 + React + Inertia + Tailwind + TypeScript. Kerjakan modular, tunggu approval setiap modul, jangan membuat fitur di luar dokumentasi.
