# Responsive Layout Reference (WAJIB)

File referensi yang saya lampirkan bukan hanya menjadi referensi visual.

AI wajib menganalisis dan memahami pola responsive yang digunakan pada referensi tersebut.

Yang harus dianalisis meliputi:

- Tata letak setiap section
- Perubahan layout dari Desktop ke Mobile
- Urutan setiap section
- Jarak antar section
- Grid system
- White space
- Alignment
- Padding
- Margin
- Ukuran card
- Komposisi card
- Hierarki informasi

# Card Summary (WAJIB)

Section Ringkasan Keuangan terdiri dari:

- Total Pemasukan
- Total Pengeluaran
- Sisa Saldo

AI harus mempelajari bagaimana section tersebut berubah pada berbagai ukuran layar.

Contoh yang harus dianalisis:

Desktop

3 Card berada dalam satu baris.

Tablet

Menyesuaikan agar tetap proporsional.

Mobile

Card berubah menjadi layout yang nyaman dibaca menggunakan lebar penuh (full width) atau pola lain yang tetap mengikuti referensi.

AI tidak boleh membuat layout responsive berdasarkan asumsi sendiri.

Gunakan pola responsive dari file referensi sebagai inspirasi, kemudian sesuaikan dengan identitas project Masjid Darus Sa'adah.

---

# Layout Behavior

AI wajib menganalisis bagaimana:

- Navbar berubah ketika mobile
- Hero berubah ketika mobile
- Card Summary berubah ketika mobile
- Grafik berubah ketika mobile
- Riwayat Transaksi berubah ketika mobile
- Footer berubah ketika mobile

Semua perubahan layout tersebut harus dipelajari dari file referensi.

Namun implementasinya harus dibuat ulang agar sesuai dengan project ini.

---

# Responsive Priority

Prioritas utama adalah:

1. Kenyamanan membaca

2. Kemudahan navigasi

3. Kecepatan menemukan informasi

4. Konsistensi layout

5. Kemudahan penggunaan menggunakan satu tangan pada perangkat mobile

---

# Jangan Menyalin Secara Identik

AI tidak boleh:

- Copy layout 100%
- Copy warna
- Copy icon
- Copy ilustrasi
- Copy typography
- Copy komponen

AI hanya boleh mengambil inspirasi mengenai:

- Struktur responsive
- Pola layout
- UX
- Hierarki informasi
- Grid system
- Spacing
- Perubahan layout Desktop → Tablet → Mobile

Kemudian AI wajib membuat desain baru yang lebih modern dan disesuaikan dengan identitas Masjid Darus Sa'adah.
